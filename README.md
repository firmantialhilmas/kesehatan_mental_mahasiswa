# Sistem Pendukung Keputusan Kesehatan Mental Mahasiswa

Aplikasi ini merupakan Sistem Pendukung Keputusan (SPK) berbasis web/mobile yang digunakan untuk membantu menentukan tingkat kondisi kesehatan mental mahasiswa menggunakan metode Naive Bayes.

## Tujuan Aplikasi
Membantu mahasiswa dalam mengenali kondisi kesehatan mental berdasarkan indikator tertentu, serta memberikan rekomendasi sebagai langkah awal penanganan.

## Konsep Sistem
Sistem ini bekerja dengan mengolah data kuesioner yang diisi oleh pengguna, kemudian dianalisis menggunakan metode Naive Bayes untuk menentukan tingkat kondisi kesehatan mental.

## Fitur Utama
- Login pengguna
- Screening kesehatan mental berbasis kuesioner
- Analisis tingkat kondisi (Rendah, Sedang, Tinggi)
- Implementasi metode Naive Bayes untuk klasifikasi
- Visualisasi hasil dalam bentuk grafik
- Riwayat hasil screening
- Rekomendasi berdasarkan hasil analisis
- Penyimpanan data secara lokal

## Metode yang Digunakan
Metode yang digunakan dalam sistem ini adalah **Naive Bayes**, yaitu metode klasifikasi berbasis probabilitas yang digunakan untuk menentukan tingkat kondisi kesehatan mental berdasarkan data yang dimasukkan oleh pengguna.

## Alur Sistem
1. Pengguna mengisi kuesioner
2. Sistem mengolah data sebagai fitur
3. Dilakukan perhitungan probabilitas menggunakan Naive Bayes
4. Sistem menentukan tingkat kondisi kesehatan mental
5. Hasil dan rekomendasi ditampilkan kepada pengguna

## Teknologi
